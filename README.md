# Piggy

Piggy is a mobile budgeting and expense tracking app built with React Native, TypeScript, and Expo, featuring SQLite for offline data storage, interactive financial charts, and spending insights.  
It helps users visualize spending patterns, track expenses by category, and understand financial habits through progress bars and pie charts.

## Features

- Category-based expense tracking
- Visual progress bars and pie charts for spending insights
- Cross-platform UI running on iOS and Android
- Local storage using SQLite for offline functionality
- Online data synchronization with Firebase
- Interactive financial charts and summaries
- Responsive and touch-friendly UX

---

## Built With

- Frontend: React Native, TypeScript
- Framework: Expo
- Database: SQLite, Firebase
